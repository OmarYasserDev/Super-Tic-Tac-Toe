# Super Tic-Tac-Toe
A modern implementation of **Ultimate Tic-Tac-Toe** built with **HTML, CSS, and JavaScript**.
## About
Super Tic-Tac-Toe is an advanced version of the classic Tic-Tac-Toe game. Instead of playing on a single board, players compete across **nine interconnected boards** arranged in a 3×3 grid.
Every move determines where the opponent must play next, making the game far more strategic than traditional Tic-Tac-Toe.
## Features
*Classic Ultimate Tic-Tac-Toe rules
*9 interactive Tic-Tac-Toe boards
*Two-player gameplay
*Automatic winner detection
*Winner popup with restart option
*Responsive and modern UI
*Smooth hover effects and animations

##  Technologies Used
* HTML5
* CSS3
* JavaScript (Vanilla)
##  Getting Started
1. Clone the repository:
```bash
git clone https://github.com/OmarYasserDev/Super-Tic-Tac-Toe.git
```
2. Open the project folder.
3. Launch `index.html` in your browser.
No installation or additional dependencies are required.
##  How to Play
* Player **X** starts the game.
* Each move sends the opponent to a specific small board.
* Win a small board by getting **three marks in a row**.
* Win the game by claiming **three small boards in a row** on the main board.
* If a target board is already completed or full, the next player may choose any available board.
